# SNU_CED_2024_Teleop

이 프로젝트는 Dynamixel 모터와 HC-05 블루투스 모듈, OpenRB-150 보드를 이용하여 로봇팔을 원격 조작하는 시스템을 구현합니다.
Leader Manipulator와 Follower Manipulator로 구성되어 있습니다.
